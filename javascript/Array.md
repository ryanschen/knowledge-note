<details>
  <summary>pop()</summary>

  * 从数组中删除最后一个元素，并返回该元素的值。<br>
  * 此方法更改数组的长度。

  ```js
  var arr = [1,2,3]
  arr.pop() // 3
  arr // [1, 2]
  ```
</details>

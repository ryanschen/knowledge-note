全局内置工具类型是对自定义的类型的再次封装，返回封装后的类型。

* Partial<T>
  该工具类型设置为是 T 类型的子集。

* Readonly<T>
  将原本 T 类型的所有属性设置为 Readonly。

* Record<K,T>
  将原本 T 类型的所有属性设置为满足 K 类型。
  ```ts
  const modules: Record<string, object> = {} // or const modules: { [key: string]: object } = {}
  ```

* Pick<T,K>
  从 T 中取一组 K 类型，组成新的类型。
  k 为 'a' | 'b'，则被定义的对象的属性需包含 'a'、'b' 属性。
